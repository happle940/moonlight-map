**白月光.skill 把这些不公平的记忆摊开，分拣，归档。**

不扮演 ta，不替 ta 说话，不帮你脑补"其实 ta 也喜欢你"。

做的事情更实在：把你脑子里关于这个人的碎片——一个场景、一句没说出口的话、一首歌、一段沉默——整理成一份看得懂的结构。把"真正发生了什么"和"你想象中发生了什么"分开。

做完之后你可能会发现：你想的不是 ta。你想的是那个还相信"喜欢就够了"的自己。

那也没关系。知道自己在想什么，总好过不知道。

## 你会拿到什么

走完一轮对话，你拿到一份**月光档案**：

```
+--------------------------------------------------+
| 月光档案 / Moonlight Dossier                     |
| 代号: 走廊尽头的人                               |
+--------------------------------------------------+
| 一句真话:                                        |
| 你忘不掉的不是 ta，是"差一点就会发生"            |
| 的那种人生感觉。                                 |
+--------------------------------------------------+
| 吸引层: 被看见感、克制的距离感                   |
| 投射层: 你把短暂的被理解放大成了                 |
| 长期的被懂得                                     |
| 未完成: 没说出口的话 → 持续反刍                  |
+--------------------------------------------------+
| 当时的你: 我以为喜欢就够了                       |
| 后来的你: 我遗憾自己什么都没说                   |
| 现在的你: 我怕下一段关系也活在对比里             |
+--------------------------------------------------+
| 一句结论: 该纪念，但别再拿 ta 当标尺             |
| 衡量新的人。                                     |
+--------------------------------------------------+
```

还有一封**未寄出的信**（可选）——不是为了寄出去，是为了让那些话别一直在你脑子里转。

## 什么时候会用到

深夜突然想起某个人，情绪像后台程序一样自己启动。听到一首歌、路过一个地方，整个人不太对。要开始新关系了，但总觉得心里还有个旧影子。

不是想复合。只是想搞明白。

---

## 跟前任.skill 的区别

前任.skill 让 ta 回来和你说话。白月光.skill 让你看清——你一直在想的到底是 ta，还是另一种人生。

前任卖"让 ta 回来"。白月光卖"让雾散开"。
