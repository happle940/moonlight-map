# 白月光.skill

> *你以为你在想 ta，也可能你在想另一种人生。*

&nbsp;

前任.skill 让 ta 回来和你说话。

白月光.skill 让你看清——

**为什么 ta 一直没从你脑子里下线。**

不是把 ta 蒸馏成 AI。是把你对 ta 的投射，蒸馏成结构。

[安装](#安装) · [使用](#使用)

---

深夜。一首歌随机播到。你没在意。

但某个画面突然回来了。走廊里 ta 回头看你。毕业那天你没说出口的话。朋友圈那首歌你偷偷听了 27 遍。

你以为你在想 ta。

**但也许你想的，是那个还相信"喜欢就够了"的自己。**

&nbsp;

白月光.skill 不替 ta 说话。不帮你脑补"其实 ta 也喜欢你"。不把执念包装成宿命。

它做的事情更实在：把你脑子里关于这个人的碎片——一个眼神、一首歌、一段沉默——整理成一份月光档案。

把"真正发生了什么"和"你想象中发生了什么"分开。

把一团雾，变成一个能命名、能归档、能放下的东西。

**不是把 ta 叫回来，是把你自己看清楚。**

---

## 你会拿到什么

聊完一轮，你会拿到一份月光档案。长这样：

```
+--------------------------------------------------+
| 月光档案 / Moonlight Dossier                     |
+--------------------------------------------------+
| 代号: 小鹿                                       |
+--------------------------------------------------+
| 一句真话:                                        |
| 你怀念的不是 ta，是那个还相信              |
| "喜欢就够了"的自己。                          |
+--------------------------------------------------+
| 吸引层: 被看见感 · 克制感                      |
| 投射层: 你把短暂的被理解放大成长期的被懂得   |
| 未完成: 没说出口 -> 持续反刍                  |
+--------------------------------------------------+
| 现实: 你们确实有过让你记很久的瞬间               |
| 投射: 你把这些瞬间扩写成了一种完整关系           |
| 不确定: ta 当时怎么想，你并不知道               |
+--------------------------------------------------+
| 时间剖面                                         |
| 当时: 我以为喜欢就够了                           |
| 后来: 我遗憾什么都没说                           |
| 现在: 我怕下一段关系活在对比里                   |
+--------------------------------------------------+
| 一句结论:                                        |
| 这不是深情未了，是一段未归档的投射关系。       |
+--------------------------------------------------+
```

不是一段空泛安慰。是可以截图、可以复看、能一眼看懂的结构。

到最后，如果你明显卡在"有些话没说出口"，它还会帮你整理一封**未寄出的信**。

不是为了寄出去。是为了让它别一直在你脑子里转。

---

## 跟前任.skill 不一样

前任.skill 让 ta 回来，用 ta 的语气和你说话。
白月光.skill 不让 ta 回来，而是帮你搞明白——为什么 ta 一直在你脑子里占着缓存位。

前任卖的是**重逢**。白月光卖的是**坐标**。

一个让你再听 ta 说句话。一个让你看清这段情绪结构上长什么样。

如果你只是想找个人扮演 ta——前任.skill 更适合你。
如果你想把一团说不清的雾整理成看得懂的东西——那就对了。

## 使用

直接说人话就行：

```
我忘不了一个人
```

```
我有个白月光
```

```
听到一首歌突然想起 ta
```

```
我想在开始新关系前清理一下旧情绪
```

你只要给它 2-4 个碎片——一个场景、一句话、一首歌、一个你现在最卡住的问题。再给 ta 起个代号。真名不是必需品。

不想深聊？说 **"快速模式"**，5 分钟过一遍。

## 安全边界

9 条，写死了，不会变：

永远不扮演 · 不编造想法 · 不鼓励纠缠 · 不美化执念 · 不把投射伪装成事实 · 危机兜底 · 最小化识别信息 · 未成年人边界 · 不做心理诊断

---

## 不只是前任

白月光不只是那个没在一起的人。

也可以是：一个已故的长辈，那些没来得及说的话。一个失联的挚友，为什么比很多现役朋友还清晰。一个影响过你的老师，你怀念的是 ta 还是那时的可能性。

---

## 写在最后

人的记忆有一种奇怪的偏好。

你记不住昨天中午吃了什么，但四年前的某个下午，ta 在走廊里回头看了你一眼，你记得清清楚楚。光线从哪个角度打进来，ta 穿的什么颜色，旁边有没有人。

这不公平。

白月光.skill 做的事情很简单：把这些不公平的记忆摊开，分拣，归档。

不是为了忘记。是为了让它别再以一团雾的形式卡着你。

做完之后你可能会发现：你想的不是 ta。你想的是那个还相信"喜欢就够了"的自己。

那也没关系。

知道自己在想什么，总好过不知道。
