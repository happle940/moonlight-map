# 白月光.skill

> 把一个忘不掉的人，整理成一份能看懂的档案。
> 不是删档。是归档。

**前任.skill 让 ta 回来和你说话。**
**白月光.skill 让你看清，为什么 ta 一直没从你脑子里下线。**

---

## 这是什么

`白月光 | Moonlight Map` 不是角色扮演 skill。
它不替 ta 说话，不帮你脑补"其实 ta 也喜欢你"，也不把执念包装成宿命。

它做的事更实在：

- 把你脑子里关于这个人的**碎片记忆**整理出来
- 把"**真正发生了什么**"和"**你想象中发生了什么**"分开
- 把"忘不掉"拆成结构：**吸引 / 投射 / 未完成**
- 最后给你一份能复看、能命名、能归档的**月光档案**

一句话：

**不是把 ta 蒸馏成 AI。是把你对 ta 的投射蒸馏成结构。**

---

## 它适合什么时刻

你大概率会在这些时刻点开它：

- 深夜突然想起某个人，情绪像后台程序一样自己启动
- 听到一首歌、路过一个地方，整个人不太对
- 要开始新关系了，但总觉得心里还有个旧影子
- 不是想复合，只是想搞明白：**我到底在想 ta，还是在想另一种人生？**

如果你只是想找个人扮演 ta，陪你演一段对话，
这个 skill 不适合你。

如果你想把一团说不清的雾，整理成一个看得懂的结构，
那就对了。

---

## 它不做什么

白月光最核心的就是：**未知**。

所以这个 skill 明确不做这些事：

- 不扮演 ta
- 不替 ta 回话
- 不编造 ta 的想法、态度、遗憾或喜欢
- 不帮你设计偶遇、试探、追回、越界联系
- 不把执念美化成"念念不忘必有回响"

**我们做的不是续写，是整理。**

---

## 你会拿到什么

完成一次完整使用后，你会拿到一份 **月光档案 / Moonlight Dossier**。

主件是：

### 月光图谱

把这段情绪拆成几个层次：

- **吸引层**：你到底被什么击中了
- **投射层**：你把什么意义放大了
- **未完成层**：到底哪件事没关掉
- **现实 / 投射 / 不确定**：哪些是真的，哪些是你脑内续写的
- **时间剖面**：当时的你、后来的你、现在的你，有什么变了
- **一句结论**：这段情绪该怎么看、该怎么放

### 未寄出的信（可选）

如果你明显卡在"有些话没说出口"，它会在最后给你整理一封：

**不是为了寄出去。**
**是为了让它别一直在你脑子里转。**

## 与前任.skill 的区别

| 维度 | 前任.skill | 白月光.skill |
|------|-----------|-------------|
| 核心卖点 | 还原 ta、让 ta 回来和你说话 | 整理你自己的投射与未完成 |
| 主要体验 | 角色扮演 / 记忆复现 | 结构分析 / 月光归档 |
| 输出形式 | 对话感、陪伴感 | 图谱感、坐标感 |
| 用户期待 | "ta 会怎么说" | "我到底在想什么" |
| 风险点 | 容易沉浸在幻想回应 | 明确切断幻想，强调边界 |
| 本 skill 态度 | 做续写 | 不做续写，只做整理 |

更直白一点：

**前任.skill 卖的是"让 ta 回来"。**
**白月光.skill 卖的是"让雾散开"。**

---

## 为什么它可能比读一百个故事更有用

因为白月光最难受的地方，往往不是"难过"。

而是：

**你总觉得只有自己这样。**

你以为你在想 ta。
也可能你在想：

- 当时那个版本的自己
- 一种没有发生的人生
- 一种被看见、被理解、但从未真正落地的关系幻觉

所以这个 skill 不提供情绪消费。
它提供的是：

**坐标。**

不是"好感人的故事"。
而是"原来这段情绪，结构上长这样"。

---

## 适合谁

- 有一个多年忘不掉的人，但并不一定真的在一起过
- 明知事情过去了，某些画面还是会反复扎一下
- 不想复合，只想搞清楚
- 想进入新关系前，先把旧投射整理干净
- 比起被安慰，更想被看清

## 不适合谁

- 只想让 AI 扮演 ta
- 只想获得"ta 其实也喜欢你"的幻想回应
- 想用它设计偶遇、试探、追回、越界联系
- 当前情绪极度失控，只需要即时安抚而不是结构整理

## 使用方式

直接用自然语言开始就行。比如：

```
我忘不了一个人
```

```
我有个白月光
```

```
听到一首歌突然想起 ta
```

```
我想知道我怀念的是 ta 还是以前的自己
```

```
我想在开始新关系前清理一下旧情绪
```

如果你不知道怎么开口，也没关系。
你只要先给它 2-4 个碎片：

- 一个场景
- 一句话，或者一段沉默
- 一首歌 / 一个物件 / 一张照片
- 你现在最卡住的问题

再给 ta 起个代号。真名不是必需品。

不想深聊？说 **"快速模式"**，5 分钟过一遍。

---

## 安全边界

这个 skill 的边界写得很死，而且不会被后续流程覆盖。

核心原则：

- 永远不扮演白月光
- 不编造对方的想法
- 不鼓励纠缠
- 不美化执念
- 不把投射伪装为事实
- 危机兜底（建议寻求现实帮助）
- 最小化识别信息（默认用代号）
- 未成年人/权力不对等边界
- 不做心理诊断

---

## 写在最后

人的记忆有一种奇怪的偏好。

你记不住昨天中午吃了什么，但四年前的某个下午，ta 在走廊里回头看了你一眼，你记得清清楚楚。光线从哪个角度打进来，ta 穿的什么颜色，旁边有没有人。

这不公平。

白月光.skill 做的事情很简单：把这些不公平的记忆摊开，分拣，归档。

不是为了忘记。是为了**让它别再以一团雾的形式卡着你**。

做完之后你可能会发现：你想的不是 ta。你想的是那个还相信"喜欢就够了"的自己。

那也没关系。

知道自己在想什么，总好过不知道。
